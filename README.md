# Aastha Narula Portfolio Site

Static GitHub Pages portfolio website for Aastha Narula.

## Source Inputs

- `../aastha-narula.pdf` - CV PDF published as the downloadable resume.
- `../Aastha_AI_Portfolio_Execution_Plan.md` - Portfolio roadmap and positioning.
- `../Aastha_Narula_CV_v4.md` - Text source for CV copy where the PDF text is not extractable locally.

## Publish

This site is static. GitHub Pages can serve it directly from the repository root.

## Local Preview

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Public Safety

- Phone number is intentionally omitted.
- The résumé PDF is available through `assets/aastha-narula.pdf`; the public page uses LinkedIn as the only visible contact path.
- Copy avoids employer-confidential details beyond public, résumé-level claims.
